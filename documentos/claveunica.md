# Integración ClaveÚnica

Una vez que ya tengas instalado SIMPLE OnPremise, debes incorporar las credenciales de Clave Única desde la sección de manager `/manager/cuentas`, seleccionando la cuenta a la que quieras asociar tus credenciales, desde el apartado "Editar ClaveÚnica" dónde podrás ingresar tanto el "`Client ID`" como el "`Client Secret`".

![Ejemplo registrar credenciales de CU desde Manager](https://github.com/digital-gob-cl/simple2/blob/master/documentos/imagenes/form_claveunica_manager.jpg)

## Nota:
En caso de que seas parte de una institución privada,  debes verificar si puedes utilizar la integración de Clave Única y posteriormente, cumplir exitosamente el proceso de activación.
Para obtener las credenciales sandbox y de producción: https://claveunica.gob.cl/instituciones
